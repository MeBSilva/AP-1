# AP 1 - SMD0045 LINGUAGEM DE PROGRAMAÇÃO SCRIPT

Projeto Cliente Servidor que modela o problema do menor valor único.

## Instruções

### Mudar o diretório para o do servidor

```bash
cd back
```

### Instale as Dependências:

```bash
yarn
```

### Inicie o Server:

```bash
yarn start:dev
```
